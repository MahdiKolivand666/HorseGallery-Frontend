# عکس‌های دسته‌بندی

در این پوشه باید 4 عکس با فرمت `.webp` قرار دهید.

## فایل‌های مورد نیاز:

1. **categories1.webp** - دسته اول (بالا سمت راست)
2. **categories2.webp** - دسته دوم (پایین سمت راست)
3. **categories3.webp** - دسته سوم (وسط، باریک)
4. **categories4.webp** - دسته چهارم (سمت چپ، بزرگ)

## مشخصات عکس‌ها:

- **سایز**: 360 x 360 پیکسل
- **فرمت**: WebP
- **کیفیت**: بالا (برای نمایش در کارت‌های دسته‌بندی)

## Layout در دسکتاپ:

```
┌───────┬───┬─────────┐
│   1   │   │         │
├───────┤ 3 │    4    │
│   2   │   │ (بزرگ)  │
└───────┴───┴─────────┘
 (مربع)  (باریک) (عریض)
```

## توجه:
اگر عکس‌ها را قرار ندهید، صفحه با خطای 404 مواجه می‌شود.
